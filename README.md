Baby Name Trends Analysis 📊

🚀 Project Overview

This project analyzes baby name trends using real-world data.

🛠️ Technologies

- Python
- Pandas
- Matplotlib

📊 Features

- Data extraction from zip files
- Year-wise analysis
- Gender-based trends
- Top 100 popular names visualization
- Top 20 popular name visualization
